 System Requirements
 -------------------
 
 - CAEN V1718 USB-VME Bridge and/or CAEN A2818 PCI-CONET Board
 - CAEN digitizer board (V1724, V1720, V1721, V1731, V1740, V1751, DT5724, DT5720, DT5721, DT5731, DT5740, DT5751,
   N6724, N6720, N6721, N6731, N6740, N6751)
 - Linux O.S.
 - For compiling graphical demos and tools only: 
   - wxWidgets 2.8.10 Library (www.wxWidgets.org)
